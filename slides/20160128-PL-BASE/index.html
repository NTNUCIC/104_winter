位元運算

電腦和人類的差別在於數字是以位元表示

可以以位元為單位計算

日常生活不常見到的運算

都當成整數進行計算

且

使用 & 可以將兩個數取AND

兩位元為1才為1

其他結果則為0

或

使用 | 可以將兩個數取OR

任一位元為1則為1

位元反轉

反向

左移

右移

使用>>將數字右移指定次數

效果相當於除以2的次方數

互斥或

使用 ^ 將兩數取Xor

兩個位元相同則為0

陣列

要使用陣列需要使用指標進行操作

宣告陣列要在變數名稱後加上使用[]

陣列宣告時要指定大小

可以用{}將其元素初始化

取得型別大小

使用sizeof()傳入變數或型別可得知其位元組數

因為是運算子所以編譯時就知道

也可以用來看陣列多大

陣列範例

可以手動指定大小

也可以把元素指派好後自動知道大小

#include <stdio.h>
int main(){
	bool arr[4],table[]={true,false,false};
	char list[5]={'a','b','c','d','e'};
	printf("%d %d %d",
		sizeof(arr)/sizeof(bool),
		sizeof(table)/sizeof(bool),
		sizeof(list)/sizeof(char));
	return 0;
}

陣列與指標

指標運算

加法減法可以使用

還可以使用 ++ 遞增

以及使用 -- 遞減

注意！此兩個運算子是地雷請小心使用

地雷範例

#include <stdio.h>
int main(){
	int n=100,m;
	m=++n;
	printf("%d %d\n",n,m);
	m=n++;
	printf("%d %d\n",n,m);
}

放前面放後面竟然結果會不一樣

再次提醒

雖然在此不探討其原理

但是記得一個指令一個動作

有遞增遞減就不要指派

愛護生命遠離debugging

指標移動

	printf("%d ",*p);p++;
	printf("%d ",*p);p++;
	printf("%d ",*p);p++;
}
